# AI NOC Copilot

An AI-powered Network Operations Center (NOC) Assistant that predicts network failures and provides root cause analysis with recommended actions.

## Features

- Network Failure Prediction using Machine Learning
- Risk Score Calculation
- Root Cause Analysis
- AI Chatbot Assistant
- Recommended Actions
- Streamlit Dashboard

## Tech Stack

- Python
- Streamlit
- Scikit-Learn
- Pandas
- Joblib

## Project Structure

├── app.py
├── train_model.py
├── generate_data.py
├── network_data.csv
├── network_model.pkl
├── utils/
└── README.md
